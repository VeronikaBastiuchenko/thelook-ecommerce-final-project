# Images

This folder contains **screenshots and visuals** used in the project documentation and dashboard report.

### Includes:
- Dashboard charts (Revenue, Categories, RFM, Cohorts, Pareto)
- Screens from Looker Studio
- Visuals for README and PDF report

### Naming convention:
- `dashboard_*.png` → dashboard views  
- `cohort_*.png` → retention heatmaps  
- `rfm_*.png` → segmentation  
- `pareto_*.png` → Pareto analysis
